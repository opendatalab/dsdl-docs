### **环境依赖**

- Python3.8+

### **DSDL-SDK安装**


DSDL-SDK 提供两种安装方式：
#### 1. pip 安装

```shell
pip install dsdl
```

#### 2. 源码安装

```shell
git clone https://github.com/opendatalab/dsdl-sdk
cd dsdl
python setup.py install
```
